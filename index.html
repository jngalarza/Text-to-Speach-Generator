3 · Voz

La primera vez descarga el modelo de voz (~80 MB) y puede tardar; después queda en caché. El español de Kokoro es correcto pero menos natural que el inglés. Tiene que correr servido por HTTP (subilo a Netlify/Vercel, o local con python -m http.server); abrirlo con doble clic file:// no carga el modelo.
